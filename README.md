Spring Boot "Microservice" parental-control-service-coding-test Project
This application indicate user about watch eligibility on requested movie based on user's provided parental control level.

#How to Run

1. Clone this repository
2. Make sure you are using JDK 1.8 and Maven 3.x
3. You can build the project and run the tests by running 'mvn clean package' or 'mvn clean install'

4. Once successfully built, you can run the service by
mvn spring-boot:run

If you get error related Address already in use
or
Error: InvocationTargetException: Connector configured to listen on port 8080 failed to start ->

Go to application.yml at src/main/resources & uncomment
  port: 8081

try step 4 again.
